FormCampain

Descripción

Este proyecto se enfoca en la gestión de formularios a través de una interfaz web. Permite a los usuarios autenticarse, completar formularios y enviar la información al servidor.

Descripción del Código

Variables Globales

  • loginContainer: Referencia al elemento del contenedor de inicio de sesión.
  • formularioContainer: Referencia al elemento del contenedor de formulario.
  • salirbtn: Referencia al botón de cierre de sesión.

Funciones Principales

  • obtenerValorParametro: Obtiene el valor de un parámetro de una URL.
  • setCookie y getCookie: Funciones para establecer y obtener cookies.
  • checkLoginStatus: Verifica el estado de inicio de sesión del usuario.
  • login: Realiza la operación de inicio de sesión.

Uso

  1. Inicio de Sesión: Abre el archivo HTML en un navegador y utiliza el formulario de inicio de sesión.
  2. Completar Formulario: Llena los campos requeridos y envía el formulario.
  3. Cerrar Sesión: Utiliza el botón de cierre de sesión para salir.
